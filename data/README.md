# Data folder

This folder contains a small sample of business data used in Week 1 discovery and ROI modelling. Contact your coach for more detailed information.

## Files in this folder

- `delinquent_accounts_sample.csv` - sample overdue account data for segmentation and self-service suitability analysis
- `recovery_activity_tracker_sample.csv` - sample operational activity history showing follow-up timing and manual workflow behaviour
- `finance_assumptions_sample.csv` - starter assumptions from finance for cost and value modelling
- `stakeholder_interview_notes.csv` - structured notes from stakeholder conversations for JTBD and change analysis

## How to use these files

- Use them as evidence, not as perfect truth.
- Add source and confidence labels when you copy values into your ROI model.
- If your recommendation uses an assumption that is not in these files, record it clearly in the assumptions tab.

## Tip

If you feel tempted to invent numbers too early, pause and see whether one of these files already gives you a baseline or at least a defensible starting point.
